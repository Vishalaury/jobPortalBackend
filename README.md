# MERN Job Portal Application

A full-stack job portal application built using the MERN stack (MongoDB, Express, React, Node.js). This platform allows users to browse jobs, apply for positions, and allows admins to manage companies, job postings, and applicants.

---

## 🔗 Live Links

* Frontend: http://job-portal-frontend-sable-chi.vercel.app
* Backend API: https://jobportalbackend-0e4n.onrender.com
* frontend Repo: https://github.com/Vishalaury/jobPortalFrontend

## ⚙️ Tech Stack

**Frontend**

* React.js (Vite)
* Tailwind CSS
* Shadcn UI
* Redux Toolkit

**Backend**

* Node.js
* Express.js
* MongoDB (Mongoose)

**Other Tools**

* JWT Authentication
* Multer (File Upload)
* REST APIs

---

## ✨ Features

### 👤 User

* Signup/Login with authentication
* Browse and search jobs
* Apply for jobs
* Update profile and upload resume

### 🛠 Admin

* Admin dashboard
* Create and manage companies
* Post and manage jobs
* View applicants
* Update application status

## ⚡ Installation

### Clone

git clone https://github.com/Vishalaury/jobPortalBackend

### Install

cd backend
npm install

cd ../frontend
npm install

### Environment Variables (.env in backend)

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key

### Run

npm run dev
